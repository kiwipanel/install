## Reporting a vulnerability

First off, thank you for taking the time to improve Kiwipanel! We genuinely appreciate your help in identifying issues. By reporting vulnerabilities, you play a huge part in keeping the project safe for everyone.

## How to Report
To keep our users safe, we kindly ask that you do not open public GitHub issues for security-sensitive bugs. Disclosing a vulnerability publicly before we have a fix ready puts existing Kiwipanel deployments at risk.

Instead, please give us a heads-up privately so we can address the issue first.

Found a bug? Here is how to reach us:

Send an email to `vuong@kwipanel.org`

Please include as much detail as you can (logs, steps to reproduce, etc.) so we can squash the bug as quickly as possible. We will get back to you promptly!
